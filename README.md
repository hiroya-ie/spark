# エレベーターピッチ

0→1のアイデアを出してグループワークを先に進めたい、という課題を解決する、  
グループワークでのプロダクト決めに苦戦するie生向けの  
無限のアイデア出しを実現するアプリ「スパーク」です。  
開発を始めたいけど作るものが決まらない、意見も全然でない、そんなときに  
キーワードをどんどんと出してアイデア出しの助けになることができます。  
既存のアプリと違って、ジャストフィットなアイデアと出会うことができます。
